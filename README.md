# 🖥️ site-exemplo

Usado na disciplina para testar comandos git e outras configurações.


---

## 🚀 Como rodar o projeto

Siga as etapas abaixo para executar o site no seu computador:

### 1. Instalar o servidor HTTP

Você precisa ter o **Node.js** instalado.
Depois, instale o pacote `http-server` globalmente com o seguinte comando:

```bash
npm install -g http-server
```

### 2. Executar o servidor

No terminal, navegue até a **pasta raiz do projeto** (onde está o arquivo `index.html`) e execute:

```bash
http-server
```

### 3. Acessar no navegador

Após executar o comando, o terminal mostrará o endereço local do servidor, algo como:

```
http://127.0.0.1:8080
```

Abra esse link no seu navegador para visualizar o site.

---

## 💡 Observações

* Se quiser alterar a porta padrão, use o parâmetro `-p`.
  Exemplo:

  ```bash
  http-server -p 3000
  ```
